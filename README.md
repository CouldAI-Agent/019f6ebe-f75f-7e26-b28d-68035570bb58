# Chess King

Chess King is a feature-rich chess platform with voice chat, events, ranking systems, tasks, and diamond-based economy. 

## Features
- **Authentication**: Social logins (Google, Apple, etc.) with OTP support.
- **Onboarding**: Profile, auto-generated ID, country selection.
- **Home Dashboard**: Profile status, Level progression, Diamond economy, PVP entry, Tasks, and Rankings.
- **Event Center**: Dynamic event listing with categories (Bonus, Recommend).
- **Task System**: Daily tasks to earn Diamonds and Level points.
- **Rankings**: Weekly reset leaderboard for top 100 users with rewards.
- **PVP Chess**: 1v1 matchmaking with bets (50/100 diamonds), real-time game board, voice chat, and gifting.

## Tech Stack
- Flutter for cross-platform app (iOS, Android, Web)
- Clean, responsive UI with bubble-styled buttons

## Setup
1. Ensure Flutter is installed.
2. Run `flutter pub get`
3. Run `flutter run`
